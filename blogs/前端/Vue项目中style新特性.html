    Vue项目中style新特性

    CoderTao 2023/2/1 vue前端

    # 1、深度选择器 :deep()用法

    当第三方组件库不满足需求的时候，则需要更改组件样式

    /* 样式穿透 */
:deep(.el-tabel) {
    background-color: red;
}

/* 或使用 /deep/ 和  >>> 语法 */
/deep/ .el-tabel {
    background-color: red;
}

>>> .el-tabel {
    background-color: red;
}

    # 2、插槽选择器 :slotted()用法

    当第三方组件库不满足需求的时候，则需要更改组件样式

    :slotted(.el-tabel) {
    background-color: red;
}

    # 3、全局选择器 :global() 用法

    在之前我们想加入全局样式通常都是新建不加 scoped 属性的 style 标签去写全局样式

    // =================== 之前 =======================
<style>
.el-tooltip__popper {
    max-width: 300px;
}
</style>

<style lang="scss" scoped>
/* 此处省略.... */
</style>

// =================== 现在 =======================

<style lang="scss" scoped>
:global(.el-tooltip__popper) {
    max-width: 300px;
}
</style>

    • 2、插槽选择器 :slotted()用法
    • 3、全局选择器 :global() 用法
